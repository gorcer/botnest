# Nest.JS + CCXT Trading Bot

## Requirements:
1. API keys from one of supported in Ccxt exchanges
2. PostgreSQL Server
3. Node v16.14.0


## How to run:


```
git clone ...
```

```
copy .env.example to .env
```


Fill settings in .env (BOT_TEST = false for production)

```
npm i
npm run execute trade
```

Enjoy


## Run test

```
npm test
```
